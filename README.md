
Hey, I’m F.A.
A self-taught coder and electronics enthusiast who loves building things from scratch. Here’s my story:

How It Started
At 9 years old, I dove into programming with p5.js, learning from Daniel Shiffman’s The Coding Train. I created fractal art, interactive drawing tools, and even a recursive tree generator—just for fun!

Hardware Adventures
Two years later, I picked up an Elegoo Uno R3 kit and followed tutorials to build projects like:

An IR remote decoder that displayed messages on the Serial Monitor.

A tilt-sensitive alarm that triggered LEDs and buzzers.

When I tried a Raspberry Pi car kit, I hit roadblocks (literally—broken buttons!), but it taught me troubleshooting the hard way.

Leveling Up
I combined existing Arduino code to create a standalone Temperature & Humidity Meter—my first fully independent project. Later, I upgraded it with ESP32 and MQTT, using AI to debug the sensor readings.

Why It Matters
Every project taught me something new: creativity with code, problem-solving with hardware, and persistence when things fail. This portfolio is where I share those wins (and lessons).
